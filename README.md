Key's PhotoBook is a frontend web-based photobooth application that allows users to capture photos directly from the browser, arrange them into a photo strip, and customize them using draggable stickers. The final composition can be downloaded as a digital image. This project focuses on DOM manipulation, event handling, and building interactive visual experiences. A modern UI, smooth animations, and intuitive user interaction were prioritized, with no backend implementation.

**Features**
-Browser-based photo capture
-Photo strip generation
-Draggable and customizable stickers
-Downloadable final image
-Interactive and aesthetic user interface

**Tech Stack**
-HTML
-CSS
-JavaScript
-local apis

**Goal**
To practice and strengthen frontend development skills through an interactive application, focusing on client-side logic, UI design, and user experience.
